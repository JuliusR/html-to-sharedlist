# HTML to sharedlist

## Environment settings

Datei `./env.php` mit folgendem Inhalt anlegen:

```
<?php

$kind_sources = array(
   'TAB1' => [ // maximal 6 Zeichen
       'https://url.de/abc1'
   ],
   'TAB2' => [
       'https://url.de/abc2',
       'https://url.de/abc3',
       'https://url.de/abc4'
   ]
);
```
